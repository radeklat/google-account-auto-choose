This addon automatically selects Google accounts based on URL patterns. Configure rules below:

  • Enabled: Enable or disable this specific rule
  • Name: A descriptive name for the rule
  • Match: Regex pattern to match URL of the correct Google account choice page
  • Email: The email address to automatically select
  • Auto-close: Regex pattern to match URL of a possible confirmation page for auto-closing (optional)
  • Auto-close within: Time window in seconds after account selection when auto-close should occur
Example:
URL: https://accounts.google.com/v3/signin/accountchooser?continue=https://accounts.google.com/o/saml2/continue
Match: .*/saml2/.*
Email: user@company.com
Auto-close: .*done=1
